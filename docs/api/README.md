# API Documentation

## Conventions

- Base path: `/api/v1` unless explicitly noted.
- Auth: `Authorization: Bearer <accessToken>`.
- Mobile-only routes in this codebase use `/api/mobile/...` (outside global `/api/v1` prefix).
- Errors: see [errors.md](./errors.md).
