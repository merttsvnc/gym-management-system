# AGENTS Rules

## Repository Documentation Policy

- Keep markdown docs under `/docs` only.
- Root markdown is restricted to `README.md` and `AGENTS.md` (`LICENSE` allowed if present).
- Do not create archive/history documentation folders.

## API Documentation Policy

- API docs are domain-based in `docs/api/*.md`.
- When endpoints change, update the matching domain doc in the same change.

## Completion Checks

- Before finishing docs changes, verify there are no broken markdown links.
